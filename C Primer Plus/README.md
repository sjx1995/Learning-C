- 第二章：<br>
	练习基本输出

- 第三章：<br>
	介绍关键字<br>
	介绍`size()`运算符<br>
	介绍数据类型<br>

- 第四章：<br>

	`&&`、`||`、`?`运算符<br>

	`break`、`continue`、`goto`跳转语句<br>
	头文件`<ctype.h>`提供字符分析函数<br>
	
- 第八章：<br>
	输入、输出和缓冲输入、缓冲输出的区别<br>
	通过键盘模拟文件结尾条件(EOF)<br>
	通过`重定向`完成对文件的读取和写入<br>
	创建更友好的用户界面，验证输入避免用户错误输入带来的问题<br>
